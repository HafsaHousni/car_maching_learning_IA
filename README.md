# Car Price Prediction  Machine Learning

## Description
Ce projet utilise des techniques de Machine Learning pour prédire le prix d'une voiture en fonction de plusieurs caractéristiques comme la marque, le modèle, l'année, le kilométrage et la couleur.

L'objectif est de construire un modèle intelligent capable d'estimer le prix d'un véhicule avec une bonne précision.

## Technologies utilisées
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Dataset
Le dataset contient plusieurs informations sur les voitures :

- Marque
- Modèle
- Année
- Kilométrage
- Type de carburant
- Couleur
- Prix

Les données sont nettoyées et préparées avant l'entraînement du modèle.

## Méthodologie

1. Chargement du dataset
2. Nettoyage et préparation des données
3. Analyse exploratoire des données
4. Encodage des variables catégorielles
5. Séparation des données (Train / Test)
6. Entraînement du modèle de Machine Learning
7. Évaluation des performances du modèle

## Modèles utilisés
- Linear Regression
- Random Forest Regressor
- Decision Tree Regressor

## Résultats
Le modèle permet de prédire le prix d'une voiture avec une bonne précision en fonction de ses caractéristiques.
